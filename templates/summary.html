Predicting Final Grades with Machine Learning

By Leah Stucky, Lakshmi Prasanna Rangam, Ju A Han, Jake Andrasovsky

Data Set

The data used for this project was collected from a study done on 2 Portuguese schools (reference below). The Data was made up of 31 attributes and 3 performance measurements. Our group applied several machine learning models towards the data in the hopes of predicting final year scores(G3).

Data Transformation

Originally, we used all 31 attributes of the data to test on the machine learning model. This method had lower than expected data test scores and predicting student performance would be difficult. At this point we decided to make two transformations to the data set. The first was to focus on the top 5 weighted attributes which we determined from random forest classifier and the second was to focus the final scores into 4 performance groups rather than the original 20 scores.

Grade transformation to performace groups

Machine Learning Models

Logistic Regression

The first model we tested on the data set was Logistic Regression and was ultimately the model saved and used for our application. After some data transformation and retesting we chose this model because it scored higher than the other models tested.

Decision Tree and Random Forest

The second model we tested the data on was SKlearn's Decision Tree combined with Random Forest. This model scored slightly less Logistic Regression but was helpful in transforming our data set by determining feature importance. We used feature importance to narrow down our data set parameters to the top 5 weighted attributes.

K Nearest Neighbors

The K Nearest Neighbors had solid test scores and would have been fine to use had we not gone with the Logistic Regression model. This model begins to elbow out at 9 groups as seen in the graph.

K Nearest Neighbors

Deep Learning

The last model we attempted was the Sequential model from TensorFlow. The accuracy of this model was not as high as the other models and we choose not to use this model in our application.
